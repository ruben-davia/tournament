# AI Skills

These playbooks help an AI agent use the framework correctly.

Use them as operating instructions:

1. identify the task
2. open the matching skill
3. gather the required inputs
4. produce the expected output

Recommended order for a new tournament:

1. `01_understand_tournament.md`
2. `02_design_scoring_rules.md`
3. `03_prepare_probability_inputs.md`
4. `04_model_the_field.md`
5. `05_run_strategy_simulation.md`
6. `06_explain_recommendations.md`
7. `07_adapt_to_new_tournament.md`
